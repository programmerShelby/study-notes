class :
    -创建类class 栗子：创建一个明星类
        -class Star{
            //内部构造函数,共有属性放入constructor
            constructor(uname,age){
                this.uname = uname;
                this.age = age;
                }
            //创建函数
            sing(song){
                console.log(this.uname + song);
            }
        }
    -利用类创建对象 new
        -var ldh = new Star('刘德华',58);
    -调用方法
        -ldh.sing('笨小孩');//刘德华笨小孩
    -类的继承 extends
        -class Father{
            constructor(x,y){
                this.x = x;
                this.y = y;
            }
            sum(){
                console.log(this.x + this.y);
            }
        }

        class Son extends Father{
            constructor(x,y){
                //super 调用父类中的构造函数
                super(x,y);
            }
        }
        var pigge = new Son(1,2);
        pigge.sum();//3

    注意：
        1.通过class 关键字创建类，类名需要首字母大写
        2.类里面有 constructor 函数，可以接受传递来的参数，同时返回实例对象
        3.constructor 函数，只要 new 生成实例时，就会自动调用这个函数，
            如果不写这个函数，类也会自动生成
        4.在生成实例时 new 不能省略
        5.语法必须规范：
            -创建类时类名不加小括号
            -生成实例时类名后面加小括号
            -构造函数时不需要加function
            -在类里面多个函数之间不需要添加逗号分隔
        6.继承中，实例化子类输出一个方法，采用 就近原则
        7.在ES6 中类没有变量提升，所以必须先定义类，才能通过类实例化对象
        8.类里面的共有属性和方法一定加this 使用
        9.constructor 里面的this 指向对象实例，方法里面的this 指向调用方法者
